# Interest Rate Models: Vasicek, CIR, and HJM Frameworks

This project provides a comprehensive analysis of interest rate modeling, covering both the theoretical derivations and the practical implementation of stochastic processes for bond pricing.

## Project Overview
Developed as part of the **MSc in Financial Risk and Data Analysis**, this project explores the foundations of quantitative financial modeling applied to interest rates. It combines a theoretical "Short Paper" with a Python-based implementation of the models.

### Models Covered:
1. **Vasicek Model**: Analysis of the mean-reverting short rate process and derivation of zero-coupon bond prices.
2. **Cox-Ingersoll-Ross (CIR) Model**: Exploration of non-negative short rate dynamics and square-root processes.
3. **Heath-Jarrow-Morton (HJM) Framework**: Theoretical foundations for modeling the entire forward rate curve.

## Repository Structure
- `SHORT_PAPER_quant_merged.pdf`: Complete theoretical report including mathematical derivations, parameter sensitivity analysis, and practical implications.
- `Group_8_CODES.ipynb`: Jupyter Notebook containing Python simulations, parameter calibration, and visualization of interest rate paths.

## Technologies Used
- **Language**: Python
- **Libraries**: NumPy, Matplotlib (for stochastic path simulations and data visualization).
